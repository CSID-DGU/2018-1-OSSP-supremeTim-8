## 게임 실행 방법
서버는 고정IP 사용 OR 하마치 사용
서버로 접속 -> 리듬게임 버튼 클릭 -> 시작하기 -> 1P 리듬게임 모드
                    -> 시작하기 -> 1P 모드          
                    -> 다른 플레이어가 클라이언트로 접속 -> IP/PORT 같도록 설정 -> 2P 모드(플레이어 추가 가능) -> 서버가 시작하기 버튼 클릭
