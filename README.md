## 게임 실행 방법
> 서버는 고정IP 사용이 필요합니다.

1. 랜선 연결 혹은 고정 IP 설정

2. 리눅스 하마치를 이용

 * 하마치 설치 : https://openwiki.kr/tech/hamachi_for_linux_command_line_version

  ![](./_etc/socket1.PNG)

 * 빨간 박스 쳐저 있는 주소를 클라이언트 생성 시 주소를 입력할 때 사용
  
  
> 게임 실행 과정

- 무조건 서버를 만들고나서 게임 종류 선택
- 리듬게임을 선택하고 시작하기를 누르면 1p 리듬게임 모드
- 그냥 시작하기를 누르면 1p 모드
- 클라이언트가 접속하고 시작하기를 누르면 멀티플레이
